# goBoard
This is a goBoard project:

Requare: must instill npm, nodejs

Start the project:
	1) first time ONLY, do "npm install -g @angular/cli"
	2) go to goBoard folder
	3) npm install
	4) npm start

For websocket server:
	1) go to server folder under goBoard folder
	2) npm instll
	3) node server.js &
